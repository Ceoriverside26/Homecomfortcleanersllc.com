## 🌐 Live Website Setup (GitHub Pages)

To make your site live:

1. Upload all files to your GitHub repository  
2. Go to **Settings → Pages**  
3. Under **Source**, select:  
   - Branch: `main`  
   - Folder: `/root`  
4. Click **Save**
